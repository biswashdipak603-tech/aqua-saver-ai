How it works

Aqua Saver AI isn't just a timer. It continuously monitors soil moisture levels and checks the OpenWeatherMap API for rain forecasts. If rain is predicted, the system pauses irrigation automatically, saving up to 40% more water than traditional systems.

Hardware Used

  • Controller: ESP32 Development Board (WiFi + Bluetooth)
  • Sensors: Capacitive Soil Moisture Sensor v1.2
  • Actuation: 5V Submersible Pump & 1-Channel Relay
  • Power: 5V 2A Power Supply / Solar Integration
  • Software: Arduino IDE, C++, Blynk IoT Cloud, OpenWeatherMap API
